5. Trabalhando com branchs

5.1 Criando uma nova branch
git branch nome-da-branch
// Acessando a nova branch
git checkout nome-da-branch
// Criando e acessando uma nova branch (Prefiro esse)
git checkout -b nome-da-branch

5.1 Aplicando merge em branchs
// Precisa estar na branch de destino
git merge nome-da-branch 

5.2 Visualizando todas as branchs existentes no repositório
// A branch corrente será marcada por um asterisco
git branch

Deletando uma branch
git branch -d nome-da-branch
